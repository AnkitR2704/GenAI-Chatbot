Mistral Chatbot 🤖

A simple chatbot web app built with Streamlit and LangChain MistralAI, powered by the ChatMistralAI model. Users can chat interactively in a browser interface.

Features
-Interactive web-based chat interface using Streamlit.
-Uses MistralAI large language model for responses.
-Stores API keys securely using .env locally or Streamlit Secrets in cloud deployment.
-Keeps chat history during the session.
